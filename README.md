# decodelabs_task03_Anshima

AI-Based Career Recommendation System
Overview

The AI-Based Career Recommendation System is a Python application that suggests suitable career domains based on a user's interests. It uses Natural Language Processing (NLP) techniques such as TF-IDF Vectorization and Cosine Similarity to generate personalized recommendations.

Features
User interest-based recommendations
Input validation
Top 5 career suggestions
Similarity score calculation
Recommendation history storage
Bar Chart and Pie Chart visualization
AI-powered matching using NLP techniques


Technologies Used
Python
Pandas
Scikit-Learn
Matplotlib

AI Concepts Used
TF-IDF Vectorization
Cosine Similarity
Content-Based Recommendation System
Conclusion

This project demonstrates how AI can be used to recommend career paths by analyzing user interests and matching them with relevant domains using similarity-based recommendation techniques.

Developed for DecodeLabs Project 3 – AI Recommendation Logic 🚀
